# Часть 1: Анализ бизнес-процессов ателье

## Основные бизнес-процессы:

### 1. Прием и оформление заказа
- Консультация клиента по услугам ателье
- Замеры и оценка изделия
- Определение стоимости и сроков выполнения
- Заключение договора на услуги

### 2. Производственный процесс  
- Разработка эскиза/технического задания
- Раскрой материалов
- Пошив/ремонт изделия
- Контроль качества на каждом этапе

### 3. Взаимодействие с клиентом
- Уведомление о готовности заказа
- Примерка и корректировка (при необходимости)
- Оплата услуг
- Выдача готового изделия

## Роли в системе:

### Клиент:
- Оформление заказа
- Оплата услуг
- Получение готового изделия

### Администратор:
- Прием и регистрация заказов
- Расчет стоимости
- Взаимодействие с клиентами
- Управление заказами

### Мастер-швея:
- Выполнение работ по заказу
- Отслеживание статуса выполнения
- Контроль качества
