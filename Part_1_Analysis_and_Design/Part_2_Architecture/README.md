## Диаграмма развертывания

![Диаграмма развертывания](../diagrams/deployment.png)

## Компоненты системы:

### Веб-сервер
- Обработка HTTP-запросов
- Бизнес-логика приложения
- Авторизация пользователей

### База данных (MySQL/PostgreSQL)
- Хранение данных о заказах
- Информация о клиентах  
- Учет материалов и фурнитуры

### Клиентские браузеры
- Веб-интерфейс для администраторов
- Интерфейс для мастеров-швей

## Принципы обмена данными:
- Клиент ↔ Сервер: HTTP/HTTPS протокол
- Сервер ↔ База данных: SQL-запросы через JDBC/ORM
- Формат данных: JSON/REST API

## Анализ эффективности:
- Масштабируемость: Возможность добавления новых серверов
- Надежность: Разделение компонентов повышает отказоустойчивость
- Производительность: Кэширование и оптимизация запросов к БД
